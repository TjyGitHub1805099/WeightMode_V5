实验器材:
    探索者STM32F4开发板
    
实验目的:
    演示如何将脱机下载器的UID自定义加密功能添加到用户的代码中
    
硬件资源:
    1,DS0(连接在PF9) 
    2,DS1(连接在PF10)
    
实验现象:
    本实验在正点原子-STM32F407探索者开发板的跑马灯实验上添加脱机下载器支持的UID自定义加密功能验证代码
    验证通过现象：LED1和LED2以1000ms的频率交替闪烁
    验证不通过现象：LED1和LED2以100ms的频率交替闪烁
